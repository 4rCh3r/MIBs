
		GbE2 MIB definitions

			v1.0


bt2Network.mib
Layer 3 configs/stats/info, including IP interface, gateway, and DNS.

bt2Physical.mib
Layer 2 config/stats/info, including VLAN, trunk groups, 
Spanning tree configuration, Port mirroring, and FDB table. 

bt2Switch.mib
General switch config/info, including stats, system info, 
port config, radius, NTP, syslog, traphost, TFTP, hardware
info, and stats. 

bt2trap.mib
GbE2 trap information

hpswitchpl.mib
HP MIB root file definition for BladeType2

rfc1213.mib
MIB-II for Network Management of TCP/IP based internets.
L3 IP information/data. 

rfc1215.mib
Coldstart, warmstart, linkup trap definitions.

rfc1493.mib
Bridge Traps, standard spanning tree and other L2 switch data.

rfc1643.mib
Ethernet-like MIB, L2 frame statistics and related information

rfc1757.mib
Remote Network Monitoring (RMON) MIB

rfc2037.mib
SNMP Network Management Entity MIB for use with Multiple STP
